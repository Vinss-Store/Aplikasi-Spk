  <p><strong>SPK SMART</strong> adalah aplikasi mobile berbasis Flutter yang memanfaatkan metode <strong>SMART (Simple Multi Attribute Rating Technique)</strong> untuk membantu pengguna dalam mengambil keputusan berdasarkan kriteria dan bobot yang ditentukan.</p>

## ⚙️ Instalasi & Penggunaan

<div align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212257463-4d082cb4-7483-4eaf-bc25-6dde2628aabd.gif" width="100">
</div>

### 📦 Langkah 1: Clone Repository

```bash
# Clone repository
git clone https://github.com/username/spk_smart.git

# Masuk ke direktori project
cd spk_smart
```

### 🔧 Langkah 2: Setup Dependencies

```bash
# Install semua dependencies
flutter pub get

# Bersihkan cache (opsional)
flutter clean
```

### 🔥 Langkah 3: Konfigurasi Firebase

1. **Buat project Firebase** di [Firebase Console](https://console.firebase.google.com/)
2. **Tambahkan aplikasi Android/iOS** ke project
3. **Download** file konfigurasi:
   - `google-services.json` untuk Android → `android/app/`
   - `GoogleService-Info.plist` untuk iOS → `ios/Runner/`

### 🚀 Langkah 4: Jalankan Aplikasi

```bash
# Jalankan di debug mode
flutter run

# Atau jalankan di device tertentu
flutter run -d <device_id>

# Build untuk release
flutter build apk --release
```

## 🎯 Cara Menggunakan Aplikasi

<div align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212257460-738ff738-247f-4445-a718-cdd0ca76e2db.gif" width="100">
</div>

### 1️⃣ **Login/Register**
- Buka aplikasi dan daftar akun baru atau login dengan akun existing
- Autentikasi menggunakan email dan password

### 2️⃣ **Input Kriteria**
- Tambahkan kriteria yang akan digunakan untuk pengambilan keputusan
- Tentukan bobot untuk setiap kriteria (total bobot = 100%)

### 3️⃣ **Input Alternatif**
- Masukkan alternatif yang akan dibandingkan
- Berikan nilai untuk setiap kriteria pada masing-masing alternatif

### 4️⃣ **Analisis Hasil**
- Lihat hasil perhitungan SMART dalam bentuk tabel dan grafik
- Alternatif dengan skor tertinggi adalah pilihan terbaik
